## EC2 Instances

### General notes
* Any instance type ending with `a` uses AMD Ryzen processors.
* Any instance type ending with `g` uses ARM Graviton processors.
* Any instance type with `d` in its suffix supports NVMe local storage.[^3]
* Any instance type with `n` in its suffix runs on the AWS Nitro hypervisor.
