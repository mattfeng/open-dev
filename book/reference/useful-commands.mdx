# Useful Commands

## SSH

* Open up SSH shell on already existing connection: ``<Enter>~C``
* Access a remote port locally: ``-L <local>:localhost:<remote>``

## tmux

* Switch among common layouts: ``<C-b><Space>``
* Swap with next pane (cw): ``<C-b>{``
* Swap with previous pane (ccw): ``<C-b>}``
