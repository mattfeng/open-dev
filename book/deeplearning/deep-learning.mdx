# Deep learning

### Limit the number of CUDA devices used
```bash
CUDA_VISIBLE_DEVICES=0,1
```

Or, in Python, before any other imports:

```python
import os
os.environ["CUDA_VISIBLE_DEVICES"] = "0,1"
```
